# PWA 아이콘 가이드

이 폴더에 다음 아이콘 파일들을 추가해주세요:

## 필수 파일

| 파일명 | 크기 | 용도 |
|--------|------|------|
| `pwa-192x192.png` | 192×192px | PWA 기본 아이콘 |
| `pwa-512x512.png` | 512×512px | PWA 스플래시/대형 아이콘 |

## 권장 파일

| 파일명 | 크기 | 용도 |
|--------|------|------|
| `apple-touch-icon.png` | 180×180px | iOS 홈 화면 아이콘 |
| `favicon.ico` | 32×32px | 브라우저 탭 아이콘 |
| `mask-icon.svg` | SVG | Safari 마스크 아이콘 |

## 팁

- 아이콘은 정사각형이어야 합니다
- 배경이 투명한 PNG를 권장합니다
- [Favicon Generator](https://realfavicongenerator.net/)를 사용하면 한 번에 모든 사이즈를 생성할 수 있습니다
